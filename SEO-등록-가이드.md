## 📋 검색 엔진 등록 방법

### 🔵 1. 구글 검색 등록

#### 1-1. Google Search Console 등록
1. **구글 서치 콘솔 접속**
   - https://search.google.com/search-console

2. **속성 추가**
   - "속성 추가" 버튼 클릭
   - URL: `https://gapyeong-pump.netlify.app` 입력

3. **소유권 확인**
   - **방법 1: HTML 파일 업로드** (추천)
     - 제공된 HTML 파일 다운로드
     - 홈페이지 루트에 업로드
     - "확인" 버튼 클릭
   
   - **방법 2: HTML 태그**
     - index.html의 `<head>` 태그 안에 제공된 메타 태그 추가
     ```html
     <meta name="google-site-verification" content="여기에_코드_입력" />
     ```

4. **Sitemap 제출**
   - 좌측 메뉴 > "Sitemaps" 클릭
   - `https://gapyeong-pump.netlify.app/sitemap.xml` 입력
   - "제출" 클릭

5. **색인 생성 요청**
   - 좌측 메뉴 > "URL 검사"
   - 홈페이지 URL 입력
   - "색인 생성 요청" 클릭

---

### 🟢 2. 네이버 검색 등록

#### 2-1. 네이버 서치어드바이저 등록
1. **네이버 서치어드바이저 접속**
   - https://searchadvisor.naver.com

2. **웹마스터 도구 > 사이트 등록**
   - "사이트 추가" 버튼 클릭
   - URL: `https://gapyeong-pump.netlify.app` 입력

3. **사이트 소유 확인**
   - **방법 1: HTML 파일 업로드** (추천)
     - 제공된 HTML 파일 다운로드
     - 홈페이지 루트에 업로드
     - "소유확인" 버튼 클릭
   
   - **방법 2: HTML 태그**
     - index.html의 21번째 줄 수정:
     ```html
     <meta name="naver-site-verification" content="여기에_코드_입력">
     ```

4. **사이트맵 제출**
   - "요청" 탭 > "사이트맵 제출"
   - `https://gapyeong-pump.netlify.app/sitemap.xml` 입력
   - "확인" 클릭

5. **RSS 제출 (선택사항)**
   - 블로그가 있다면 RSS 피드 URL 제출

---

## 🎯 추가 SEO 최적화 방법

### 1. 네이버 플레이스 등록
- **네이버 플레이스 등록**: https://business.naver.com
- 업체명, 주소, 전화번호 정확히 입력
- 사진 10장 이상 등록 (작업 현장, 제품 사진)
- 영업시간, 서비스 항목 상세히 입력
- **네이버 지도 검색에서 최상단 노출 가능!**

### 2. 구글 비즈니스 프로필 등록
- **Google Business Profile**: https://business.google.com
- 무료로 구글 지도, 구글 검색에 업체 정보 표시
- 고객 리뷰 관리 가능

### 3. 네이버 블로그 운영
- 정기적으로 작업 사례, 펌프 점검 팁 등 포스팅
- 포스팅마다 "가평모터수리", "가평펌프수리" 키워드 포함
- 블로그 → 홈페이지 링크 연결

### 4. 콘텐츠 추가 (권장)
- 작업 사례 페이지 추가
- 펌프 종류별 수리 가이드
- 자주 묻는 질문 확장
- 고객 후기 더 많이 추가

### 5. 모바일 최적화
- ✅ 이미 완료 (반응형 디자인)

### 6. 페이지 속도 최적화
- 이미지 압축 (.jpg, .webp 사용)
- 불필요한 스크립트 제거
